## ¿Qué es JSON?
**JSON (JavaScript Object Notation)** es un **formato ligero de intercambio de datos** utilizado para enviar y recibir información entre aplicaciones, especialmente en entornos web.  
Aunque está basado en JavaScript, es **independiente del lenguaje** y compatible con muchos otros.



## Concepto 
JSON representa la información mediante **pares clave–valor**, lo que permite organizar los datos de forma clara, estructurada y fácil de procesar.



## Características principales
- Formato ligero y compacto  
- Fácil de leer y escribir  
- Procesamiento rápido  
- Muy usado en APIs y servicios web  
- Compatible con múltiples lenguajes  
- Usa objetos `{}` y arreglos `[]`



## Comparación entre JSON, XML y HTML

**JSON**  
Se utiliza principalmente para el **intercambio de datos**. Es ligero, rápido y fácil de leer, por lo que es el formato más común en APIs modernas.

**XML**  
También se usa para el intercambio de datos, pero emplea **etiquetas** similares a HTML. Es más **pesado y verboso**, aunque permite estructuras más complejas y validaciones estrictas.

**HTML**  
No está pensado para intercambiar datos, sino para **mostrar contenido en el navegador**. Define la estructura visual de una página web.
